Detailed Description

Author
Ryan Curtin

Defines the DualTreeTraverser for the BinarySpaceTree tree type. This is a nested class of BinarySpaceTree which traverses two trees in a depth-first manner with a given set of rules which indicate the branches which can be pruned and the order in which to recurse.
